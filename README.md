# Apple Game Plus Solver Helper

`Apple Game Plus Solver Helper`는 `https://apple.oshizi.com/`에서 동작하는 브라우저 유저스크립트입니다.

게임 보드를 읽어서 합이 10이 되는 직사각형 후보를 표시하고, 추천 수를 자동으로 드래그해 빠르게 플레이할 수 있도록 도와줍니다.

## 스크립트 정보

- 이름: `Apple Game Plus Solver Helper`
- 버전: `0.1.1`
- 제작자: `Yettttie`
- 대상 사이트: `https://apple.oshizi.com/*`
- 권장 실행 환경: `Tampermonkey`

## 주요 기능

- 현재 보드에서 추천 수를 계산해 힌트 표시
- 추천 영역을 자동으로 드래그하는 `Auto Drag` 지원
- 탐색 모드 선택
  - `균형`
  - `즉시 최대`
  - `길 만들기`
- 탐색 엔진 선택
  - `휴리스틱`
  - `빔 서치`
  - `빔 + 엔드게임`
- 탐색 깊이 `1수 ~ 6수` 설정
- 안내 방식 선택
  - `실시간 재계산`
  - `계획 따라가기`
- `오프닝 계산`으로 시작 구간 미리 계산
- 단축키 `\`로 1순위 추천 수 즉시 실행

## 동작 환경

- 크롬, 엣지 등 유저스크립트 매니저 확장을 설치할 수 있는 브라우저
- 권장 확장: `Tampermonkey`
- 대체 가능: `Violentmonkey` 등 `@grant none` 기반 유저스크립트를 실행할 수 있는 확장
- 대상 사이트: `https://apple.oshizi.com/*`

## 설치 방법

### 1. 유저스크립트 매니저 설치

브라우저에 유저스크립트 매니저 확장을 먼저 설치합니다.

- 가장 안정적인 사용을 위해 `Tampermonkey`를 권장합니다.
- Chrome 웹스토어 또는 Edge 부가 기능 스토어에서 `Tampermonkey` 검색
- 설치 후 브라우저에 확장이 활성화되어 있는지 확인

### 2. 스크립트 설치

다음 두 방법 중 하나로 설치하면 됩니다.

#### 방법 A. 파일로 직접 설치

1. 저장소의 `apple-solver.user.js` 파일을 엽니다.
2. 파일 내용을 복사합니다.
3. 유저스크립트 매니저 대시보드에서 `새 스크립트 만들기`를 선택합니다.
4. 기본 내용을 모두 지우고 복사한 내용을 붙여넣습니다.
5. 저장합니다.

#### 방법 B. GitHub 업로드 후 Raw 링크로 설치

GitHub에 올린 뒤에는 `apple-solver.user.js` 파일의 `Raw` 주소로 바로 설치할 수 있습니다.

1. GitHub 저장소에 `apple-solver.user.js` 업로드
2. GitHub에서 해당 파일 열기
3. `Raw` 버튼 클릭
4. 유저스크립트 설치 화면이 열리면 `설치` 선택

## 사용 방법

### 1. 게임 페이지 접속

스크립트를 설치한 뒤 `https://apple.oshizi.com/`에 접속하면 자동으로 동작합니다.

### 2. 우측 패널 확인

페이지에 Solver Helper 패널이 나타나며 아래 항목을 조절할 수 있습니다.

- `모드`: 추천 수를 어떤 성향으로 고를지 결정
- `엔진`: 탐색 방식 선택
- `탐색 깊이`: 몇 수 앞까지 볼지 설정
- `안내 방식`: 실시간 재계산 또는 계획 기반 안내 선택

### 3. 버튼 기능

- `Auto Drag`: 현재 1순위 추천 수를 자동 실행
- `Refresh`: 현재 보드를 다시 읽고 힌트를 새로 계산
- `오프닝 계산`: 현재 보드 기준으로 시작 구간 계산 수행

### 4. 단축키

- `\` : 현재 1순위 추천 수 자동 실행

## 추천 사용 흐름

1. 게임 페이지를 엽니다.
2. 처음에는 기본 세팅인 `균형 + 휴리스틱 + 3수`로 동작합니다.
3. 계산 결과가 아쉽다면 `빔 서치` 또는 탐색 깊이를 높여봅니다.
4. 가장 유력한 수를 바로 실행하고 싶으면 `Auto Drag` 또는 `\`를 사용합니다.
5. 초반 운영을 미리 보고 싶으면 `오프닝 계산`을 눌러 확인합니다.

## 기본 세팅

스크립트를 처음 실행하면 아래 기본값으로 동작합니다.

- 기본 모드: `균형`
- 기본 엔진: `휴리스틱`
- 기본 탐색 깊이: `3수`
- 기본 안내 방식: `실시간 재계산`

## 추천 세팅

- 권장 모드: `길 만들기`
- 권장 엔진: `휴리스틱`
- 권장 탐색 깊이: `5수`

위 조합은 속도와 결과 균형이 비교적 괜찮은 추천값입니다.

참고 성능:

- `Ryzen 5 9600X`, `RAM 64GB` 환경에서 해당 설정 기준 계산에 약 `1분 30초` 정도가 소요되었습니다.
- 실제 시간은 브라우저 상태, 현재 보드, 백그라운드 프로그램, 확장 프로그램 수에 따라 달라질 수 있습니다.

## 주의 사항

- 이 스크립트는 `https://apple.oshizi.com/*`에서만 동작합니다.
- 사이트 구조가 바뀌면 힌트 표시나 자동 드래그가 동작하지 않을 수 있습니다.
- 탐색 깊이와 엔진을 높일수록 계산 시간이 길어질 수 있습니다.
- 선택한 모드, 깊이, 엔진, 안내 방식 일부 설정은 브라우저 저장소에 저장됩니다.
